# NutriPlan AI Frontend

This is the premium Next.js frontend for [NutriPlan AI](https://github.com/nihalgbailur/Nutrition-ai-agent) — an agentic meal planning app for Indian households.

## Tech Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS + custom design system
- framer-motion (rich animations)
- next-themes (beautiful light/dark mode)
- TanStack Query + Sonner toasts
- Lucide icons + custom food photography & videos

## Development

From the project root:

```bash
cd frontend
npm install
npm run dev
```

The frontend expects the FastAPI backend running on port 8000 (see root README for full instructions).

## Key Highlights

- Rich custom-generated Indian food imagery and cinematic videos
- Deep framer-motion animations (counters, staggers, agent loading flows, etc.)
- Fully featured dark mode with elegant theming
- Clean, component-driven architecture matching the agentic backend

See the [root README](../README.md) for the full project overview, agent architecture, and run instructions.
